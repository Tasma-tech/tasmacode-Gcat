# tasmacode-Gcat

## Descrição
Este projeto é uma versão gráfica do Tasmacode construída com PySide6 e widgets. Ele apresenta recursos de IA, autocompletar e múltiplos recursos.

## 🚀 Recursos Principais
- **Integração com IA** - Recursos inteligentes de análise e sugestão
- **Autocompletar** - Facilita e acelera a codificação
- **Múltiplas Fontes de Recursos** - Acesso a diversos recursos e bibliotecas
- **Interface Gráfica Moderna** - Construída com PySide6 para melhor experiência

## 📋 Instruções de Instalação

### Pré-requisitos
- Python 3.8+
- pip

### Passos de Instalação
1. Clone este repositório:
   ```bash
   git clone https://github.com/John-BrenoF/tasmacode-Gcat.git
   cd tasmacode-Gcat
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Execute o projeto:
   ```bash
   python main.py
   ```
